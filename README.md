Features
Addition, subtraction, multiplication, and division.
Supports decimals and prevents invalid inputs (like multiple decimal points or leading zeros).
Delete and clear functionality.
Overwrite previous result after evaluation.
Responsive grid layout similar to a phone calculator.
Modern design with hover effects, rounded buttons, and clear display.
